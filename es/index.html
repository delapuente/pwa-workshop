PWA Workshop

The PWA Workshop aims at providing a step-by-step guide to integrate the core technologies of the so-called PWA: Service Workers, Web Manifest and Push Notifications.

The Recommendations utility is a single-view application although it does not fit the SPA definition since there are real navigations between URLs even when they present the same view.

Parts of this series:

  1. PWA Workshop @ step 1
  2. PWA Workshop @ step 2
  3. PWA Workshop @ step 3
  4. Recommendations
